Configuração do Ambiente

Alguns dos tutoriais neste site são fornecidos para várias combinações de linguagens de programação e IDEs. Com base na sua preferência, configure uma das opções seguintes ambientes de desenvolvimento em sua máquina:

  • Node.js & VSCode
  • .NET 6 & VSCode
  • .NET 6 & VS2022

Interface de linha de comando​

A interface de linha de comando, uma CLI, é uma ferramenta essencial no desenvolvimento de aplicações web modernas. Ele geralmente é usado para inicializar novos projetos, lidar com suas dependências ou para gerenciar seus código fonte. Diferentes sistemas operacionais têm suas próprias opções (por exemplo, no Windows há o Prompt de Comando e PowerShell, e sistemas baseados em Unix têm todos os tipos de shells) Mas, no nosso caso, vamos usar o 'bash', que está disponível cross a platform.

  • Em sistemas baseados em Unix, 'bash' (ou um de seus sabores como 'zsh') provavelmente está disponível já, e se não, você pode obtê-lo de https://www.gnu.org/software/bash.
  • No Windows, você pode instalar o Git para Windows que vem empacotado com um terminal 'bash'

Gerenciamento de código-fonte​

Outra parte essencial do desenvolvimento de software moderno, especialmente para web, é o código fonte Ferramenta de Gestão Git. Vá para o Downloads e instale a versão mais recente para sua plataforma. Como mencionado na seção anterior, o instalador do Windows também vem com a CLI 'bash'.

Para verificar se o 'git' está instalado e disponível, tente executar o seguinte comando em 'bash':

git --version

Tempo de execução​

Também precisaremos do Node.js runtime para executar nosso código e do gerenciador de pacotes Node.js, NPM, para gerenciar dependências de terceiros 3rd. Você pode obter um instalador para sua plataforma em https://nodejs.org/en/download que irá instalar ambos.

Para verificar se as ferramentas estão disponíveis, tente executar os dois comandos a seguir em 'bash':

node -v
npm -v

Você deve ver algo assim:

Editora​

Para a codificação em si, você pode usar o editor que mais gostar. Se você estiver procurando recomendações, recomendamos Visual Studio Code Como é gratuito, multi-plataforma, rápido e embalado com muitos recursos excelentes. Basta instalar a compilação estável mais recente para a sua plataforma.

tip

Criamos uma extensão para o Visual Studio Code que fornece informações sobre alguns dos serviços e dados da APS diretamente do editor: [Ferramentas do Autodesk Forge] (https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools). Isso pode ser útil ao depurar seus próprios aplicativos APS.

Then you need to install C# Extension (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for C# and click install as shown in the image below.
