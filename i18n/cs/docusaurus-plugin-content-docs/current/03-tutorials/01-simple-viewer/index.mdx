# Jednoduchý prohlížeč

## Úvod

V tomto tutoriálu vytvoříme jednoduchou APS aplikaci, která nám umožní
nahrávejte, překládejte a prohlížejte 3D návrhy a 2D výkresy.

<Image img={PreviewUrl} />

Využijeme následující komponenty platformy:

- [Ověření](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview)
- [Správa dat](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview)
- [Derivát modelu](https://aps.autodesk.com/en/docs/model-derivative/v2/developers_guide/overview)
- [Prohlížeč](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview)



:::tip

Pokud nemáte zájem sestavovat aplikaci od nuly a chcete si ji jen vyzkoušet
a spusťte jej lokálně, kompletní implementace je k dispozici na GitHubu:
- Pro Node.js: https://github.com/autodesk-platform-services/aps-simple-viewer-nodejs
- Pro .NET: https://github.com/autodesk-platform-services/aps-simple-viewer-dotnet


:::
