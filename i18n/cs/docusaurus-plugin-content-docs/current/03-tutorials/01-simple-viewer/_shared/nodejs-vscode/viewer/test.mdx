A to je vše! Vaše přihláška je nyní kompletní. Spusťte (nebo restartujte) aplikaci z editoru Visual Studio Code obvyklým způsobem,
a v prohlížeči přejděte na [http://localhost:8080](http://localhost:8080). Mělo by vám být předloženo
jednoduché uživatelské rozhraní s rozevíracím seznamem v pravém horním rohu, které se nakonec naplní všemi modely
K dispozici v kontejneru vaší aplikace a s tlačítkem pro nahrávání nových modelů. Jakmile vy
vyberte jeden z názvů z rozevíracího seznamu, odpovídající model se načte do prohlížeče, který zabírá
zbytek webové stránky.
