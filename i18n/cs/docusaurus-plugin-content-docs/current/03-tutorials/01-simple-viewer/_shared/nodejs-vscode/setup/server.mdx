V kořenovém adresáři složky projektu vytvořte soubor server.js s následujícím kódem:

```js title="server.js"
const express = require('express');
const { PORT } = require('./config.js');
let app = express();
app.use(express.static('wwwroot'));
app.listen(PORT, function () { console.log(`Server listening on port ${PORT}...`); });
```

Prozatím server moc nedělá, jenom obsluhuje statické prostředky z podsložky wwwroot
(která je momentálně prázdná) a hlásí všechny chyby do konzoly a zpět klientovi.

Dále přidáme skript '"start": "node server.js"' do souboru 'package.json', abychom mohli
Snadno spusťte naši aplikaci později:

```json
// ...
"scripts": {
    "start": "node server.js"
}
// ...
```
