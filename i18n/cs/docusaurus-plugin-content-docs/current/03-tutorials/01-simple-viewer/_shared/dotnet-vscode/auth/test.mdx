Pokud je aplikace stále spuštěna, restartujte ji (například pomocí _Run > Restartovat Debugging_,
nebo kliknutím na zelenou ikonu restartu), v opačném případě jej znovu spusťte (pomocí _Run > Start Debugging_,
nebo stisknutím klávesy 'F5'). Když přejdete na [http://localhost:8080/api/auth/token](http://localhost:8080/api/auth/token)
v prohlížeči by teď měl server odpovědět objektem JSON obsahujícím data přístupového tokenu.
