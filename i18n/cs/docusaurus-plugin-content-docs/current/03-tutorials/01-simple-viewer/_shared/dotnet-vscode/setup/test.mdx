Spusťte aplikaci z editoru Visual Studio Code (například prostřednictvím _Run > Start Debugging_
nebo stisknutím klávesy F5) a přejděte na položku [http://localhost:8080](http://localhost:8080)
v prohlížeči. Měli byste dostat odpověď 404, protože jsme ještě neimplementovali žádnou logiku serveru.
To bude cílem dalšího kroku.
