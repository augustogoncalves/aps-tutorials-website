### Instalace balíčků

Ve výchozím nastavení je projekt Node.js prázdný, takže musíme nainstalovat několik balíčků pomocí npm install. Začněme základním expresním serverem, body-parserem pro zpracování JSON, multerem pro nahrávání souborů a samozřejmě APS.



:::caution

Spusťte jednu instalaci npm najednou.


:::



```bash
npm install express --save
npm install express-formidable --save
npm install multer --save
npm install cookie-session --save
npm install forge-apis --save
npm install autodesk.forge.designautomation --save
npm install body-parser --save
npm install form-data --save
npm install socket.io --save
```

> Parametr --save označuje, že modul by měl být zahrnut do souboru package.json jako závislost.

Nakonec otevřete package.json a do "skriptů" přidejte "start": "node start.js", řádek. Nyní by vaše složka měla mít složku "node_modules" a vaše "package.json" by měla vypadat takto

```json title=package.json
{
  "name": "designAutomationSample",
  "version": "1.0.0",
  "description": "",
  "main": "start.js",
  "scripts": {
    "start": "node start.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "MIT",
  "dependencies": {
    "autodesk.forge.designautomation": "^3.0.3",
    "body-parser": "^1.19.0",
    "cookie-session": "^2.0.0",
    "express": "^4.17.1",
    "forge-apis": "^0.9.2",
    "form-data": "^4.0.0",
    "multer": "^1.4.5-lts.1",
    "socket.io": "^4.0.1"
  }
}
```

> Číslo verze (např. forge-apis 0.9.2) se může lišit, při vytvoření tohoto kurzu se jednalo o nejnovější verzi.

### Soubory a složky

Chcete-li vytvořit novou složku nebo soubor, klikněte pravým tlačítkem myši na oblast "Průzkumník" vlevo a vyberte Nová složka nebo Nový soubor.

Vytvořte složku /routes/ pro všechny soubory na straně serveru a složku /wwwroot/' pro všechny soubory na straně klienta.

V tomto okamžiku by váš projekt měl mít následující strukturu

<img src={VsCodeFolderExplorerImg} style={{ width: 300 }} />

Tento soubor označuje Visual Studio Code, jak bychom měli spustit náš projekt. Přejděte do nabídky Spustit >> Přidat konfiguraci... a v okně Vybrat prostředí, které se zobrazí nahoře, zvolte Node.js. Do vytvořeného souboru /.vscode/launch.json zadejte následující

```json title=launch.json
{
  // Use IntelliSense to learn about possible attributes.
  // Hover to view descriptions of existing attributes.
  // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "Launch Program",
      "program": "${workspaceFolder}/start.js",
      "env": {
        "APS_CLIENT_ID": "your id here",
        "APS_CLIENT_SECRET": "your secret here"
      }
    }
  ]
}
```



:::note


It's important to define ID & Secret as environment variables so our project can later be deployed online. More on this later, in Deployment.


:::
