Projekt .NET Core ASP.NET, který jsme vytvořili v Visual Studio, je prázdný, proto se při jeho spuštění nic nespustí. Vzhledem k tomu, že náš projekt designAutomationSample nyní obsahuje úvodní stránku na wwwroot/index.html, měli bychom vidět jeho obsah při spuštění.

Visual Studio by měl vypsat všechny prohlížeče, které jsou v systému k dispozici, abyste si mohli vybrat jeden pro spuštění projektu. Stačí vybrat jeden a kliknout na ikonu "přehrát". Prohlížeč by se měl otevřít s úvodní stránkou.
