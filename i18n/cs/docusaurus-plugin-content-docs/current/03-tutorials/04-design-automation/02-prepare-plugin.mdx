# Vytvořit zásuvný modul

Automatizace návrhu používá ".bundle" stejně jako [Autodesk App Store](https://www.autodesk.com/developer-network/app-store), což znamená, že musíte vytvořit PackageContents.xml a ZIP s knihovnou DLL (a dalšími požadovanými soubory). Podrobné informace o tom, jak je vytvořit, naleznete v Centru pro vývojáře Autodesk App Store.

V této sekci vytvoříme základní plugin, který aktualizuje parametry width a height a výsledný soubor uloží. Také podpůrné soubory (PackageContents.xml) a strukturu složek pro jejich umístění. Nakonec vytvořte soubor .ZIP připravený k nahrání do automatizace návrhu.

V kořenové složce vytvořte složku "bundles".

<img src={VsCodeFolderExplorerImg} style={{ width: 300 }} />

### Předpoklady

- 7zip: slouží k vytvoření .ZIP se soubory svazků, nainstalujte prosím odtud. Tento výukový program předpokládá, že 7zip je nainstalovaný ve výchozí složce: C:\Program Files\7-Zip\7z.exe.

### Další předpoklady

Pro další relaci můžete použít předem sestavený modul plug-in. Nebo pokud se rozhodnete jej postavit, budete potřebovat

- **Visual Studio**: Vyžaduje se Visual Studio 2019 nebo novější, navštivte [tento odkaz](https://visualstudio.microsoft.com/vs/).

- **AutoCAD, Inventor, Revit nebo 3ds Max**: Chcete-li vyvíjet, testovat a ladit zásuvný modul pro automatizaci návrhu: [AutoCAD](https://www.autodesk.com/products/autocad/overview) | [Vynálezce](https://www.autodesk.com/products/inventor/overview) | [Revit](https://www.autodesk.com/products/revit/overview) | [3ds Max](https://www.autodesk.com/products/3ds-max/overview).

---
