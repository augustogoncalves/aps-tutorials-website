# Automatizace designu

## Úvod

Tento kurz vás provede vytvořením webové aplikace, která může nahrát vstupní soubor, změnit jeho parametr "šířka" a "výška" a uložit výstupní soubor. Uživatelské rozhraní vám umožní definovat aktivitu sady prostředků AppBundle & (jako počáteční konfiguraci) a spouštět pracovní položky s výstupním výsledkem.

> Vstupní i výstupní soubory jsou uloženy v OSS Buckets, můžete použít [Simple Viewer](../simple-viewer/) pro jejich zobrazení.
