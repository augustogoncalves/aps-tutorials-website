## Vyzkoušejte si to

A to je ze strany serveru vše. Je čas to vyzkoušet!

Spusťte (nebo restartujte) aplikaci z editoru Visual Studio Code obvyklým způsobem, 
- Použijte ID (odstranění **b.**) jednoho rozbočovače ACC, jak jste se dostali v předchozím kroku v adrese: http://localhost:8080/api/admin/projects?accountId={your-account-id}.
V takovém případě by serverová aplikace měla odpovědět seznamem JSON všech projektů
K dispozici v zadaném rozbočovači.
- Použijte ID projektu, jak jste se dostali v předchozím kroku, zkuste zavolat na adresu: 
http://localhost:8080/api/admin/project/users?projectId={your-project-id}, serverová aplikace by měla
odpovězte seznamem JSON všech uživatelů z tohoto projektu.



:::info

Můžete získat různý seznam projektů podle 2 různých koncových bodů, zde je vysvětlení:
- S rozhraním Data Management API vrátí všechny projekty, kterých se aktuální uživatel účastní.
- Rozhraní API pro správu účtu vrátí všechny projekty v účtu, pokud je aktuální uživatel správcem účtu.


:::
