Spusťte aplikaci ze sady Visual Studio (například prostřednictvím _Debug > Start Debugging_
nebo stisknutím klávesy F5) a automaticky se otevře [http://localhost:8080](http://localhost:8080)
ve vašem prohlížeči. Měli byste dostat odpověď 404, protože jsme ještě neimplementovali žádnou logiku serveru.
