# Ověřování

V tomto kroku rozšíříme implementaci serveru tak, aby se mohl sám ověřit
k platformě APS, provést uživatele [3nohým pracovním postupem OAuth](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token),
a generování přístupových tokenů pro různé potřeby.



:::tip

Osvědčeným postupem je vygenerovat "interní" token s více možnostmi (například
umožňující vlastníkovi vytvářet nebo odstraňovat soubory ve službě Správa dat), které budou použity pouze
serverem a "veřejný" token s menším počtem funkcí, se kterými lze bezpečně sdílet
logika na straně klienta.


:::
