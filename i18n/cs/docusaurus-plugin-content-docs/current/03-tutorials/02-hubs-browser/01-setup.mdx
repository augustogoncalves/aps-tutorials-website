## Konfigurace aplikace

přihlašovací údaje naší aplikace APS pro komunikaci se službami Autodesk Platform Services nebo adresa URL zpětného volání
kam budou naši uživatelé přesměrováni během [3-legged authentication workflow](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token/).
