Pro samotné kódování můžete použít kterýkoli editor, který se vám nejvíce líbí.
Pokud hledáte doporučení, doporučujeme [Visual Studio Code](https://code.visualstudio.com)
protože je zdarma, multiplatformní, rychlý a nabitý spoustou skvělých funkcí.
Jednoduše nainstalujte nejnovější stabilní sestavení pro vaši platformu.



:::tip

Vytvořili jsme rozšíření pro Visual Studio Code, které poskytuje přehled o
některé služby APS a data přímo z editoru:
[Nástroje Autodesk Forge](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
To může být užitečné při ladění vlastních aplikací APS.


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

Toto rozšíření se používá pro vývoj webových aplikací v jazyce C# v editoru Visual Studio Code:
[Rozšíření C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp).
To může být užitečné při nastavování a ladění vlastních APS aplikací.


:::
