V našich kurzech budeme používat Visual Studio Community 2022. Můžete si stáhnout _Visual Studio Installer_
z [zde](https://visualstudio.microsoft.com/vs/community/) a poté nainstalujte požadovanou verzi a edici
sady Visual Studio z instalační aplikace:

![Visual Studio Installer](vs-installer.webp)

Ujistěte se, že jste zahrnuli _ASP.NET a webovou development_ komponentu a zvažte zahrnutí
komponentu _.NET Desktop development_, pokud plánujete vyvíjet pluginy pro desktopové aplikace
jako je Revit nebo Inventor.

![Visual Studio Components](vs-components.webp)
