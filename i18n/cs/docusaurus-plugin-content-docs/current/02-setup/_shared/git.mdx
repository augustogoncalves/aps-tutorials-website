Další nezbytnou součástí moderního vývoje softwaru, zejména pro web, je zdrojový kód
Nástroj pro správu [git](https://git-scm.com). Přejděte na stránku [Soubory ke stažení](https://git-scm.com/downloads)
a nainstalujte nejnovější verzi pro vaši platformu. Jak je uvedeno v předchozí části,
instalační program systému Windows je dodáván také s rozhraním příkazového řádku 'bash'.

Chcete-li zkontrolovat, zda je 'git' nainstalován a dostupný, zkuste v 'bash' spustit následující příkaz:

```bash
git --version
```
