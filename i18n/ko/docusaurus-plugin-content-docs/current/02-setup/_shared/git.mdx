현대 소프트웨어 개발, 특히 웹의 또 다른 필수적인 부분은 소스 코드입니다
관리 도구 [git](https://git-scm.com). [다운로드](https://git-scm.com/downloads)로 이동합니다.
섹션을 열고 플랫폼에 대한 최신 릴리스를 설치합니다. 이전 섹션에서 언급했듯이
Windows 설치 프로그램에는 'bash' CLI도 함께 제공됩니다.

'git'이 설치되어 있고 사용 가능한지 확인하려면 'bash'에서 다음 명령을 실행해 보십시오.

```bash
git --version
```
