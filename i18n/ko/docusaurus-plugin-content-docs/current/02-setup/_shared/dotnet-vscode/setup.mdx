## 런타임

또한 코드를 실행하고 제3자 종속성을 관리하기 위해 .NET 6 _runtime_이 필요합니다.
https://dotnet.microsoft.com/download/dotnet/6.0 에서 플랫폼에 대한 설치 프로그램을 얻을 수 있습니다.



:::info

샘플 애플리케이션은 .NET 6.0으로 개발 및 테스트되었습니다.


:::



도구를 사용할 수 있는지 확인하려면 'bash'에서 다음 명령을 실행해 보십시오.

```bash
dotnet --version
```
