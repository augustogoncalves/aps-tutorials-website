명령줄 인터페이스인 _CLI_는 최신 웹 애플리케이션을 개발할 때 필수적인 도구입니다.
새 프로젝트를 초기화하거나, 종속성을 처리하거나, 프로젝트를 관리하는 데 자주 사용됩니다.
소스 코드. 운영 체제마다 고유한 옵션이 있습니다(예: Windows에는
[명령 프롬프트](https://en.wikipedia.org/wiki/Cmd.exe) 및 [PowerShell](https://en.wikipedia.org/wiki/PowerShell)
유닉스 기반 시스템에는 모든 종류의 [쉘](https://en.wikipedia.org/wiki/Unix_shell))이 있습니다.
그러나 우리의 경우에는 _cross-platform_ 사용할 수 있는 'bash'를 사용할 것입니다.

- 유닉스 기반 시스템에서는 'bash'(또는 'zsh'와 같은 버전 중 하나)를 사용할 수 있습니다.
이미 그렇지 않은 경우 https://www.gnu.org/software/bash 에서 얻을 수 있습니다.
- Windows에서는 번들로 제공되는 [Git for Windows](https://gitforwindows.org)를 설치할 수 있습니다.
