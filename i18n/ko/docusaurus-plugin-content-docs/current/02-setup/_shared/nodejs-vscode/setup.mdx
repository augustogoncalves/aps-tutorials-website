## 런타임

또한 코드를 실행하려면 Node.js _runtime_이 필요하고 Node.js 패키지 관리자인 _NPM_이 필요합니다.
제3자 종속성을 관리합니다. https://nodejs.org/en/download 에서 플랫폼에 대한 설치 프로그램을 얻을 수 있습니다.
둘 다 설치합니다.

도구를 사용할 수 있는지 확인하려면 'bash'에서 다음 두 명령을 실행해 보십시오.

```bash
node -v
npm -v
```

다음과 같은 내용이 표시되어야 합니다.
