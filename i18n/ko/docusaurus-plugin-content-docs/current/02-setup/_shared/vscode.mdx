코딩 자체의 경우 가장 좋아하는 편집기를 사용할 수 있습니다.
권장 사항을 찾고 있다면 [Visual Studio Code](https://code.visualstudio.com)를 사용하는 것이 좋습니다.
무료이고 크로스 플랫폼이며 빠르며 많은 훌륭한 기능으로 가득 차 있습니다.
플랫폼에 맞는 안정적인 최신 빌드를 설치하기만 하면 됩니다.



:::tip

다음에 대한 인사이트를 제공하는 Visual Studio Code용 확장을 빌드했습니다.
일부 APS 편집기에서 직접 서비스 및 데이터 :
[Autodesk Forge 도구](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
이는 자체 APS 응용 프로그램을 디버깅 할 때 유용 할 수 있습니다.


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

이 확장은 Visual Studio Code 편집기에서 C#으로 웹 애플리케이션을 개발하는 데 사용됩니다.
[C# 확장](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp).
이는 자체 APS 응용 프로그램을 설정하고 디버깅할 때 유용할 수 있습니다.


:::
