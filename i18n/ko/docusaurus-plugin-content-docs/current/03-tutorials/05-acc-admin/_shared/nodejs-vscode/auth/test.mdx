응용 프로그램이 계속 실행 중인 경우 응용 프로그램을 다시 시작합니다(예: _Run > 다시 시작 Debugging_
또는 녹색 재시작 아이콘을 클릭하여), 그렇지 않으면 다시 시작하십시오(_Run > 시작 Debugging_ 사용,
또는 'F5'를 눌러).

[http://localhost:8080/api/auth/login](http://localhost:8080/api/auth/login)로 이동할 때
브라우저에서 Autodesk 로그인 페이지로 리디렉션되어야 하며 로그인 후
지금은 단순히 'Cannot GET /'을 표시하는 응용 프로그램으로 다시 리디렉션되어야합니다.
이는 'GET /' 엔드포인트를 아직 구현하지 않았기 때문에 예상됩니다. 그러나
브라우저 개발 도구를 사용하여 'localhost' 출처에 대해 브라우저에 저장된 쿠키를 탐색합니다.
응용 프로그램이 이미 인증 데이터를 저장하고 있음을 알 수 있습니다.
