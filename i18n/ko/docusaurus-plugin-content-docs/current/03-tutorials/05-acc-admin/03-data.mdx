## 사용해 보기

평소와 같이 Visual Studio Code에서 앱을 시작(또는 다시 시작)하고
[http://localhost:8080/api/hubs](http://localhost:8080/api/hubs)를 클릭합니다.
서버는 액세스 권한이 있는 모든 허브의 JSON 목록으로 응답해야 합니다.
허브 중 하나의 ID를 복사하고 다른 주소(http://localhost:8080/api/hubs/your-hub-id/projects)에서 사용해 보세요.
이 경우 서버 애플리케이션은 모든 프로젝트의 JSON 목록으로 응답해야 합니다
지정된 허브에서 사용할 수 있습니다.



:::info

이전 단계에서 로그인 절차를 건너뛰었거나 서버 애플리케이션을 다시 시작한 경우
[http://localhost:8080/api/auth/login](http://localhost:8080/api/auth/login)로 이동해야 할 수도 있습니다.
테스트하기 전에 모든 인증 데이터를 쿠키에서 사용할 수 있는지 다시 확인합니다.
'/api/hubs' 엔드포인트입니다.


:::





:::tip

Google Chrome을 사용하는 경우 [JSON 포맷터](https://chrome.google.com/webstore/detail/json-formatter/bcjindcccaagfpapjjmafapmmgkkhgoa?hl=en)를 설치하는 것이 좋습니다.
또는 JSON 응답의 형식을 자동으로 지정하는 유사한 확장입니다.


:::
