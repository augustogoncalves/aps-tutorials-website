# 활동 정의

활동은 지정된 엔진을 사용하여 실행할 수 있는 작업의 사양입니다. 입력 및 출력 파일의 수와 사용할 AppBundle 및 진입점을 지정합니다.

이 자습서 샘플에서 활동에는 2개의 입력(파일 및 JSON 데이터)과 1개의 출력(파일)이 있습니다.
