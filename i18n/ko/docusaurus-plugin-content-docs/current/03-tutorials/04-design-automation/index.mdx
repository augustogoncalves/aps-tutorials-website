# 설계 자동화

## 소개

이 자습서에서는 입력 파일을 업로드하고, 'width' 및 'height' 매개 변수를 변경하고, 출력 파일을 저장할 수 있는 웹앱을 만드는 방법을 안내합니다. UI를 사용하면 AppBundle 및 Activity(초기 구성)를 정의하고 출력 결과와 함께 Workitems를 실행할 수 있습니다.

> 입력과 출력 파일이 모두 OSS Buckets에 저장되며, [Simple Viewer](../simple-viewer/) 튜토리얼을 사용하여 볼 수 있습니다.
