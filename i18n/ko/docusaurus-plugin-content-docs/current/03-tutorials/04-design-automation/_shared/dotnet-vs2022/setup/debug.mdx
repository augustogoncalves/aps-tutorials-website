Visual Studio에서 만든 .NET Core ASP.NET 프로젝트는 비어 있으므로 시작할 때 아무 것도 실행되지 않습니다. 이제 designAutomationSample 프로젝트에 wwwroot/index.html 의 시작 페이지가 포함되어 있으므로 시작 시 해당 콘텐츠를 볼 수 있습니다.

Visual Studio는 프로젝트를 시작하기 위해 선택할 수 있도록 시스템에서 사용할 수 있는 모든 브라우저를 나열해야 합니다. 하나를 선택하고 "재생" 아이콘을 클릭하기만 하면 됩니다. 브라우저가 시작 페이지와 함께 열려야 합니다.
