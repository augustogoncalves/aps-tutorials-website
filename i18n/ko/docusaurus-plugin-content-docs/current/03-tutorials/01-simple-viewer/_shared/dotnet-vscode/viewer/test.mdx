그리고 그게 다야! 이제 응용 프로그램을 실행할 준비가 되었습니다. 평소와 같이 명령줄에서 시작합니다.

```bash
export APS_CLIENT_ID=your-own-aps-client-id
export APS_CLIENT_SECRET=your-own-aps-client-secret
export APS_BUCKET=optional-custom-bucket-name
dotnet run
```

그런 다음 브라우저에서 [http://localhost:8080](http://localhost:8080)로 이동합니다. 당신은 그래야 합니다
오른쪽 상단 모서리에 드롭다운이 있는 간단한 UI가 표시되며 결국 채워집니다.
구성된 버킷에서 사용할 수 있는 모든 모델과 새 모델을 업로드하기 위한 버튼이 있습니다.
드롭다운에서 옵션 중 하나를 선택하자마자 해당 모델이 로드됩니다
웹 페이지의 나머지 부분을 차지하는 뷰어에서.
