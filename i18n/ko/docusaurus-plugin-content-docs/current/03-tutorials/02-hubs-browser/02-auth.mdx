# 인증

이 단계에서는 자체 인증할 수 있도록 서버 구현을 확장합니다
APS 플랫폼에서 [3-legged OAuth 워크플로](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token)를 통해 사용자를 안내합니다.
다양한 요구 사항에 대한 액세스 토큰을 생성합니다.



:::tip

더 많은 기능을 가진 "내부" 토큰을 생성하는 것이 좋습니다(예:
소유자가 데이터 관리 서비스에서 파일을 만들거나 삭제할 수 있도록 허용)
서버에 의해, 그리고 안전하게 공유할 수 있는 기능이 더 적은 "공개" 토큰
클라이언트 쪽 논리입니다.


:::
