# 허브 브라우저

## 소개

이 튜토리얼에서는 콘텐츠를 탐색할 수 있는 APS 응용 프로그램을 빌드할 것입니다
[BIM 360 Docs](https://www.autodesk.com/bim-360/construction-document-management-software/)와 같은 다른 APS 기반 응용프로그램에서 인증 된 사용자
[오토데스크 문서](https://www.autodesk.com/products/autodesk-docs/overview) 또는 [Fusion Team](https://www.autodesk.com/products/fusion-360/overview)을 클릭합니다.



:::note

[액세스 프로비저닝](../../01-getting-started/index.mdx#provision-access-in-other-products)를 ACC 또는 BIM360 계정 중 하나로 복사합니다.
응용 프로그램이 프로젝트, 폴더 및 문서를 볼 수 있도록 합니다.


:::



<Image img={PreviewUrl} />

플랫폼의 다음 구성 요소를 사용합니다.

- [인증](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview)
- [데이터 관리](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview)
- [뷰어](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview)



:::tip

응용 프로그램을 처음부터 빌드하는 데 관심이 없고 시도만 하려는 경우
로컬에서 실행하면 GitHub에서 전체 구현을 사용할 수 있습니다.
- https://github.com/autodesk-platform-services/aps-hubs-browser-nodejs
- https://github.com/autodesk-platform-services/aps-hubs-browser-dotnet


:::
