# 대시보드

## 소개

이 튜토리얼에서는 [Simple Viewer](../simple-viewer) 응용 프로그램
대시보드와 유사한 기능(예: 정보 검색 및 집계)
설계 메타데이터에서 차트 및 데이터 그리드에 결과를 표시하고
이러한 새 시각적 개체와의 상호 작용을 기반으로 하는 뷰어의 상태입니다.

All the functionality in this tutorial is implemented as viewer extensions, and will only use
the viewer UI, so you're free to use any APS application you want as your starting point.
Note however that the names of folders, and the way you import JavaScript files into your
web application, might be a little bit different.



:::caution

이 자습서 코드의 일부에서는 디자인 요소에 특정 속성이 포함되어 있다고 가정합니다
_Volume_, _Level_ 또는 _Price_와 같습니다. 작업하고 싶은 특정 디자인이 있는 경우 다음을 수행할 수 있습니다.
물론 다른 속성을 사용하도록 코드를 수정하고, 그렇지 않으면 공식 속성 중 하나를 사용하는 것이 좋습니다.
[Revit 2022 샘플 프로젝트 파일](https://knowledge.autodesk.com/support/revit/getting-started/caas/CloudHelp/cloudhelp/2022/ENU/Revit-GetStarted/files/GUID-61EF2F22-3A1F-4317-B925-1E85F138BE88-htm.html)
여기서 이러한 속성은 항상 존재합니다.


:::

[Viewer](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview)를 사용합니다.
API 및 다음과 같은 다양한 기능:

- 뷰어 확장 만들기 및 로드
- 로드된 모델의 메타데이터 쿼리
- 시청자 이벤트에 대한 반응
- 뷰어 상태 제어
- 뷰어 도구 모음 사용자 지정
- 사용자 지정 UI 요소 만들기



:::tip

이 자습서의 전체 구현은 [대시보드](https://github.com/autodesk-platform-services/aps-simple-viewer-nodejs/tree/dashboards)에서 사용할 수 있습니다.
