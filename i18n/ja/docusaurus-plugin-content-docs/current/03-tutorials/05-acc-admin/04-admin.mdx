## 試してみる

サーバー側は以上です。試してみる時が来ました!

通常どおり、Visual Studio Code からアプリを起動 (または再起動) し、
- アドレスの前の手順で取得した 1 つの ACC ハブの ID(**b.** を削除) を使用します: http://localhost:8080/api/admin/projects?accountId={your-account-id}。
この場合、サーバー アプリケーションは、すべてのプロジェクトの JSON リストで応答する必要があります
指定したハブで使用できます。
- 前の手順で取得したプロジェクトIDを使用して、アドレスに電話をかけてみてください。
http://localhost:8080/api/admin/project/users?projectId={your-project-id}の場合、サーバーアプリケーションは
このプロジェクトのすべてのユーザーの JSON リストで応答します。



:::info

2つの異なるエンドポイントによって異なるプロジェクトリストを取得する場合がありますが、ここでは説明します。
- データ管理 API を使用すると、現在のユーザーが関与しているすべてのプロジェクトが返されます。
- アカウント管理APIでは、現在のユーザーがアカウント管理者の場合、アカウント内のすべてのプロジェクトを返します。


:::
