# 基本的なUI

インターフェイスは、バニラの HTML5 と 'JavaScript' に基づいています。基本的にはどのサーバー側でも同じですが、いくつかの違いがあります。 Websocket の実装では、socket.io ('Node.js') または SignalR ('.NET 6') が使用されます。
UI ('HTML' と 'JavaScript') ファイルから始めましょう。「wwwroot」フォルダの下に、次の内容の「index.html」ファイルを作成します。

<EnvTabs
  NodeJsVsCode={IndexHtmlForNode}
  DotNetVsCode={IndexHtmlForNET}
  DotNetVs2022={IndexHtmlForNET}
/>

「wwwroot」フォルダの下に「js」フォルダを作成します。次に、「wwwroot/js」フォルダー内に、次の内容の「ApsDesignAutomation.js」ファイルを作成します。
