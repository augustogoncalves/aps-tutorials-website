- Visual Studio 2022 を起動し、[新しいプロジェクトの作成] を選択します
- テンプレート検索バーに「ASP.NET Core Empty」と入力します。[次へ] を選択して選択します。
- 次のダイアログで、「designAutomationSample」という名前を付けましょう。
- 次のダイアログで、[.NET 6.0 (Long Term Support)] が選択されていることを確認し、[Configure for HTTPS] のチェックを外します。
- 「最上位レベルのステートメントを使用しない」チェックボックスをオンにして最上位レベルのステートメントの使用を無効にし、「作成」をクリックします。
- Autodesk Forge NuGet パッケージをインストールする: プロジェクト(ソリューション エクスプローラ)を右クリックし、[NuGet パッケージを管理]を選択し、[参照]で Autodesk.Forge を検索して Autodesk.Forge をインストールします。入力結果と出力結果を [OSS バケット](https://aps.autodesk.com/en/docs/data/v2/developers_guide/basics/) にアップロードします。
- 次のパッケージの .NET 6 互換バージョンもインストールしてください。



:::note

このドキュメントの執筆時点では、チュートリアルではパッケージ名の横に指定されたバージョンを使用しています。


:::



```bash
Autodesk.Forge v1.9.7
Autodesk.Forge.DesignAutomation v5.1.1
Microsoft.AspNetCore.Mvc.NewtonsoftJson v6.0.16
Microsoft.AspNetCore.SignalR.Protocols.NewtonsoftJson v6.0.16
```
