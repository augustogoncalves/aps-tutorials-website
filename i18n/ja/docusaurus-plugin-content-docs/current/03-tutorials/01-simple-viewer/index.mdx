# シンプルなビューア

## イントロダクション

このチュートリアルでは、簡単な APS アプリケーションを構築します。
3Dデザインと2D図面をアップロード、翻訳、プレビューします。

<Image img={PreviewUrl} />

プラットフォームの次のコンポーネントを使用します。

- [認証](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview)
- [データ管理](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview)
- [モデル派生](https://aps.autodesk.com/en/docs/model-derivative/v2/developers_guide/overview)
- [ビューア](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview)



:::tip

アプリケーションをゼロから構築することに興味がなく、試してみたい場合
ローカルで実行すると、完全な実装がGitHubで入手できます。
- Node.jsの場合:https://github.com/autodesk-platform-services/aps-simple-viewer-nodejs
- .NET の場合: https://github.com/autodesk-platform-services/aps-simple-viewer-dotnet


:::
