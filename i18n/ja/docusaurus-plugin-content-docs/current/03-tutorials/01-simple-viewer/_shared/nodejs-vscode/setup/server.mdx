次のコードを使用して、プロジェクト フォルダーのルートに 'server.js' ファイルを作成します。

```js title="server.js"
const express = require('express');
const { PORT } = require('./config.js');
let app = express();
app.use(express.static('wwwroot'));
app.listen(PORT, function () { console.log(`Server listening on port ${PORT}...`); });
```

今のところ、サーバーはあまり何もしておらず、「wwwroot」サブフォルダーから静的アセットを提供しているだけです
(現在は空です)、エラーをコンソールに報告し、クライアントに報告します。

次に、'"start": "node server.js"' スクリプトを 'package.json' ファイルに追加して、
後でアプリケーションを簡単に実行できます。

```json
// ...
"scripts": {
    "start": "node server.js"
}
// ...
```
