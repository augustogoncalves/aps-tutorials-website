コーディング自体については、最も好きなエディタを使用できます。
レコメンデーションをお探しの場合は、[Visual Studio Code](https://code.visualstudio.com) をお勧めします
無料で、クロスプラットフォームで、高速で、多くの優れた機能が満載です。
お使いのプラットフォーム用の最新の安定版ビルドをインストールするだけです。



:::tip

Visual Studio Code の拡張機能を構築し、次の分析情報を提供しました
一部の APS サービスとデータをエディターから直接入手します。
[Autodesk Forge ツール](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools)。
これは、独自の APS アプリケーションをデバッグする場合に役立ちます。


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

この拡張機能は、Visual Studio Code エディターで C# で Web アプリケーションを開発するために使用されます。
[C# 拡張機能](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp)。
これは、独自の APS アプリケーションを設定およびデバッグする場合に役立ちます。


:::
