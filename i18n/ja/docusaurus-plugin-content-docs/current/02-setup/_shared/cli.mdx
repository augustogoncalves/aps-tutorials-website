コマンドラインインターフェイスである_CLI_は、最新のWebアプリケーションを開発する際に不可欠なツールです。
これは、新しいプロジェクトの初期化、依存関係の処理、またはプロジェクトの管理によく使用されます。
ソースコード。さまざまなオペレーティングシステムには独自のオプションがあります(たとえば、Windowsには
[コマンドプロンプト](https://en.wikipedia.org/wiki/Cmd.exe)と[PowerShell](https://en.wikipedia.org/wiki/PowerShell)、
Unix ベースのシステムには、あらゆる種類の [シェル](https://en.wikipedia.org/wiki/Unix_shell)) があります。
しかし、私たちの場合、_cross platform_利用可能な「bash」を使用します。

- Unix ベースのシステムでは、'bash' (または 'zsh' のようなそのフレーバーの 1 つ) が利用可能である可能性が最も高いです
すでに、そうでない場合は、https://www.gnu.org/software/bash から入手できます。
- Windowsでは、同梱の[Git for Windows](https://gitforwindows.org)をインストールできます。
