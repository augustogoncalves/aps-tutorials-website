## Tempo de execução

Também precisaremos do Node.js _runtime_ para executar nosso código e do gerenciador de pacotes Node.js, _NPM_,
para gerenciar dependências de terceiros 3rd. Você pode obter um instalador para sua plataforma em https://nodejs.org/en/download
que irá instalar ambos.

Para verificar se as ferramentas estão disponíveis, tente executar os dois comandos a seguir em 'bash':

```bash
node -v
npm -v
```
