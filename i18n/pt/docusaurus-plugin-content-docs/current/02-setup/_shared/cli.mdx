A interface de linha de comando, uma _CLI_, é uma ferramenta essencial no desenvolvimento de aplicações web modernas.
Ele geralmente é usado para inicializar novos projetos, lidar com suas dependências ou para gerenciar seus
código fonte. Diferentes sistemas operacionais têm suas próprias opções (por exemplo, no Windows há
o [Prompt de Comando](https://en.wikipedia.org/wiki/Cmd.exe) e [PowerShell](https://en.wikipedia.org/wiki/PowerShell),
e sistemas baseados em Unix têm todos os tipos de [shells](https://en.wikipedia.org/wiki/Unix_shell))
Mas, no nosso caso, vamos usar o 'bash', que está disponível _cross a platform_.

- Em sistemas baseados em Unix, 'bash' (ou um de seus sabores como 'zsh') provavelmente está disponível
já, e se não, você pode obtê-lo de https://www.gnu.org/software/bash.
- No Windows, você pode instalar o [Git para Windows](https://gitforwindows.org) que vem empacotado
com um terminal 'bash'
