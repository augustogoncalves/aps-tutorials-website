Para a codificação em si, você pode usar o editor que mais gostar.
Se você estiver procurando recomendações, recomendamos [Visual Studio Code](https://code.visualstudio.com)
Como é gratuito, multi-plataforma, rápido e embalado com muitos recursos excelentes.
Basta instalar a compilação estável mais recente para a sua plataforma.



:::tip

Criamos uma extensão para o Visual Studio Code que fornece informações sobre
alguns dos serviços e dados da APS diretamente do editor:
[Ferramentas do Autodesk Forge] (https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
Isso pode ser útil ao depurar seus próprios aplicativos APS.


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

Essa extensão é usada para desenvolver aplicativos Web em C# no editor de código do Visual Studio:
[Extensão C#] (https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp).
Isso pode ser útil ao configurar e depurar seus próprios aplicativos APS.


:::
