# Autenticação

Nesta etapa, vamos estender a implementação do servidor para que ele possa se autenticar
para a plataforma APS, orientar o usuário através de um [fluxo de trabalho OAuth de 3 pernas](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token),
e gerar tokens de acesso para diversas necessidades.



:::tip

É uma boa prática gerar um token "interno" com mais recursos (por exemplo,
permitindo que o proprietário crie ou exclua arquivos no serviço de Gerenciamento de Dados) que só serão usados
pelo servidor e um token "público" com menos recursos que podem ser compartilhados com segurança
a lógica do lado do cliente.


:::
