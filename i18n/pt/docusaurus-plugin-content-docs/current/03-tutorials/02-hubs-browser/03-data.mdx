## Experimente

E é isso para o lado do servidor. Hora de experimentar!

Inicie (ou reinicie) o aplicativo do Visual Studio Code como de costume e navegue até
[http://localhost:8080/api/hubs](http://localhost:8080/api/hubs) no navegador.
O servidor deve responder com uma lista JSON de todos os hubs aos quais você tem acesso.
Tente copiar o ID de um dos hubs e usá-lo em outro endereço: http://localhost:8080/api/hubs/your-hub-id/projects.
Nesse caso, o aplicativo de servidor deve responder com uma lista JSON de todos os projetos
disponível no hub especificado.



:::info

Se você ignorou o procedimento de login na etapa anterior ou reiniciou o aplicativo do servidor,
talvez seja necessário ir para [http://localhost:8080/api/auth/login](http://localhost:8080/api/auth/login)
novamente para se certificar de que todos os dados de autenticação estão disponíveis em cookies antes de testar
o ponto de extremidade '/api/hubs'.


:::





:::tip

Se você estiver usando o Google Chrome, considere instalar o [JSON Formatter](https://chrome.google.com/webstore/detail/json-formatter/bcjindcccaagfpapjjmafapmmgkkhgoa?hl=en)
ou uma extensão semelhante para formatar automaticamente respostas JSON.


:::
