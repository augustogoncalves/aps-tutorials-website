### Instalar pacotes

Por padrão, um projeto Node.js está vazio, então precisamos instalar alguns pacotes com npm install. Vamos começar com um servidor expresso básico, body-parser para manipulação de JSON, multer para upload de arquivos e, claro, APS.



:::caution

Execute uma instalação npm de cada vez.


:::



```bash
npm install express --save
npm install express-formidable --save
npm install multer --save
npm install cookie-session --save
npm install forge-apis --save
npm install autodesk.forge.designautomation --save
npm install body-parser --save
npm install form-data --save
npm install socket.io --save
```

> O parâmetro --save indica que o módulo deve ser incluído no arquivo package.json como uma dependência.

Por fim, abra o package.json e, dentro de "scripts", adicione "start": "node start.js", linha. Agora sua pasta deve ter uma pasta 'node_modules' e sua 'package.json' deve ter esta aparência

```json title=package.json
{
  "name": "designAutomationSample",
  "version": "1.0.0",
  "description": "",
  "main": "start.js",
  "scripts": {
    "start": "node start.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "MIT",
  "dependencies": {
    "autodesk.forge.designautomation": "^3.0.3",
    "body-parser": "^1.19.0",
    "cookie-session": "^2.0.0",
    "express": "^4.17.1",
    "forge-apis": "^0.9.2",
    "form-data": "^4.0.0",
    "multer": "^1.4.5-lts.1",
    "socket.io": "^4.0.1"
  }
}
```

> O número da versão (por exemplo, forge-apis 0.9.2) pode variar, era a versão mais recente quando este tutorial foi criado.

### Arquivos e Pastas

Para criar uma nova pasta ou arquivo, clique com o botão direito do mouse na área "Explorer" à esquerda e selecione Nova pasta ou Novo arquivo.

Crie uma pasta '/routes/' para todos os arquivos do lado do servidor e uma pasta '/wwwroot/' para todos os arquivos do lado do cliente.

Neste ponto, o projeto deve ter a seguinte estrutura

<img src={VsCodeFolderExplorerImg} style={{ width: 300 }} />

Esse arquivo indica ao Visual Studio Code como devemos executar nosso projeto. Vá para o menu Executar >> Adicionar configuração... e, na janela Selecionar Ambiente que aparece na parte superior, escolha Node.js. No arquivo /.vscode/launch.json criado, digite o seguinte

```json title=launch.json
{
  // Use IntelliSense to learn about possible attributes.
  // Hover to view descriptions of existing attributes.
  // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "Launch Program",
      "program": "${workspaceFolder}/start.js",
      "env": {
        "APS_CLIENT_ID": "your id here",
        "APS_CLIENT_SECRET": "your secret here"
      }
    }
  ]
}
```



:::note


It's important to define ID & Secret as environment variables so our project can later be deployed online. More on this later, in Deployment.


:::
