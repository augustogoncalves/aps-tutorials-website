O projeto de ASP.NET do .NET Core que criamos no Visual Studio está vazio, portanto, quando o iniciamos, nada é executado. Como nosso projeto designAutomationSample agora contém uma página inicial em wwwroot/index.html então devemos ver seu conteúdo no lançamento.

Visual Studio deve listar todos os navegadores disponíveis no sistema para você escolher um para iniciar seu projeto. Basta selecionar um e clicar no ícone "play". O navegador deve abrir com sua página inicial.
