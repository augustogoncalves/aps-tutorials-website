# Definir Atividade

Atividade é a especificação de uma ação que pode ser executada usando um mecanismo especificado. Ele especifica o número de arquivos de entrada e saída, e o AppBundle e o ponto de entrada a ser usado.

Neste exemplo de tutorial, a atividade tem 2 entradas (arquivo & dados JSON) e 1 saída (arquivo).
