# Automação de Projetos

## Introdução

Este tutorial irá guiá-lo através da criação de um webapp que pode carregar um arquivo de entrada, alterar seu parâmetro 'width' e 'height' e salvar o arquivo de saída. A interface do usuário permitirá que você defina AppBundle & Activity (como uma configuração inicial) e execute Workitems, com um resultado de saída.

> Os arquivos de entrada e saída são salvos em OSS Buckets, você pode usar [Simple Viewer](.. /simple-viewer/) tutorial para visualizá-los.

<ResponsiveVideo src={PreviewUrl} />
To modify your models, you need the following steps:

Create a server << First time developer? You should start here

- Interface do usuário básica do aplicativo
- Preparar um plugin
- Definir uma atividade
- Executar item de trabalho
