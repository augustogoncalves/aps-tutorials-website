# Autenticação

Nesta etapa, vamos estender a implementação do servidor para que ele possa se autenticar
para a plataforma APS e gerar tokens de acesso para diferentes casos de uso.



:::tip

É uma boa prática gerar um token "interno" com mais recursos (por exemplo,
permitindo que você crie ou exclua arquivos no serviço de Gerenciamento de Dados) que só serão usados
pelo servidor e um token "público" com menos recursos que podem ser compartilhados com segurança
a lógica do lado do cliente.


:::
