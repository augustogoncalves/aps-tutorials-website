## Experimente

E é isso para o lado do servidor. Hora de experimentar!

Inicie (ou reinicie) o aplicativo a partir do Visual Studio Code como de costume, 
- Use o ID(removendo **b.**) de um hub ACC como você obtém na etapa anterior no endereço: http://localhost:8080/api/admin/projects?accountId={your-account-id}.
Nesse caso, o aplicativo de servidor deve responder com uma lista JSON de todos os projetos
disponível no hub especificado.
- Use o ID do projeto como você obtém na etapa anterior, tente ligar para o endereço: 
http://localhost:8080/api/admin/project/users?projectId={your-project-id}, o aplicativo de servidor deve
responder com uma lista JSON de todos os usuários deste projeto.



:::info

Você pode obter uma lista de projetos diferente pelos 2 pontos de extremidade diferentes, aqui estão os explicativos:
- Com a API de Gerenciamento de Dados, ele retornará todos os projetos que o usuário atual está envolvido.
- Com a API de Administração de Conta, ela retornará todos os projetos na conta se o usuário atual for Administrador de Conta.


:::
