## Pruébalo

Y eso es todo para el lado del servidor. ¡Es hora de probarlo!

Inicie (o reinicie) la aplicación desde Visual Studio Code como de costumbre, 
- Utilice el ID (eliminando **b.**) de un concentrador ACC como se obtiene en el paso anterior en la dirección: http://localhost:8080/api/admin/projects?accountId={your-account-id}.
En este caso, la aplicación de servidor debe responder con una lista JSON de todos los proyectos
disponible en el concentrador especificado.
- Use el ID del proyecto como lo obtuvo en el paso anterior, intente llamar a la dirección: 
http://localhost:8080/api/admin/project/users?projectId={your-project-id}, la aplicación de servidor debe
responda con una lista JSON de todos los usuarios de este proyecto.



:::info

Puede obtener una lista de proyectos diferente por los 2 puntos finales diferentes, aquí están los explicaciones:
- Con la API de gestión de datos, devolverá todos los proyectos en los que esté involucrado el usuario actual.
- Con la API de administración de cuentas, devolverá todos los proyectos de la cuenta si el usuario actual es administrador de cuentas.


:::
