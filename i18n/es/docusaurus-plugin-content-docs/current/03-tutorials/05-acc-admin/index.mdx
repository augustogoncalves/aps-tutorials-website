# Administrador de ACC

## Introducción



:::note

Solo **Node.js** está disponible para el tutorial.


:::




In this tutorial we're going to build an APS application that will let us browse Hubs and Projects
of our authenticated user in [Autodesk Construction Cloud](https://www.autodesk.com/products/autodesk-docs/overview),
list all the information of projects and project users, you can also export these information to a CSV
file easily, but most important, the tutorial will show case how to import multiple projects into a selected account
based on a template project list csv file, or import multiple users into a selected project from user list template 
csv file.



:::note

Asegúrese de [aprovisionar acceso](../../01-getting-started/index.mdx#provision-access-in-other-products) a sus cuentas de ACC
para que la aplicación pueda ver tus proyectos.


:::



<Image img={PreviewUrl} />

Utilizaremos los siguientes componentes de la plataforma:

- [Autenticación](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview)
- [Gestión de datos](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview)
- [Administrador de Autodesk Construction Cloud](https://aps.autodesk.com/en/docs/acc/v1/overview/field-guide/admin/)



:::tip

Si no está interesado en crear la aplicación desde cero y solo quiere probar
y ejecutarlo localmente, la implementación completa está disponible en GitHub:
- https://github.com/autodesk-platform-services/aps-acc-admin-tutorial-nodejs


:::
