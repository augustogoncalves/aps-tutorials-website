Inicie la aplicación desde Visual Studio (por ejemplo, a través de la _Debug > Iniciar Debugging_
o pulsando 'F5'), y se abrirá automáticamente [http://localhost:8080](http://localhost:8080)
en su navegador. Debería obtener una respuesta 404 porque aún no hemos implementado ninguna lógica de servidor.
Ese va a ser el objetivo del siguiente paso.
