## Configuración de la aplicación

Nuestra aplicación necesitará un par de parámetros de configuración para ejecutarse correctamente, por ejemplo,
las credenciales de nuestra aplicación APS para comunicarse con Autodesk Platform Services, o la URL de devolución de llamada
a dónde se redirigirá a nuestros usuarios durante el [flujo de trabajo de autenticación de 3 patas](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token/).
Pasaremos estos parámetros a la aplicación de servidor mediante variables de entorno.
