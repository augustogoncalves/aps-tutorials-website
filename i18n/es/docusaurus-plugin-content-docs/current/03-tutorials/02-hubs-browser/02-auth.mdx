# Autenticación

En este paso, vamos a extender la implementación del servidor para que pueda autenticarse
a la plataforma APS, guíe al usuario a través de un [flujo de trabajo OAuth de 3 patas](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token),
y generar tokens de acceso para diversas necesidades.



:::tip

Es una buena práctica generar un token "interno" con más capacidades (por ejemplo,
permitir al propietario crear o eliminar archivos en el servicio de gestión de datos) que solo se utilizarán
por el servidor, y un token "público" con menos capacidades que se puede compartir de forma segura con
la lógica del lado del cliente.


:::
