# Definir actividad

La actividad es la especificación de una acción que se puede ejecutar mediante un motor especificado. Especifica el número de archivos de entrada y salida, así como el AppBundle y el punto de entrada que se van a utilizar.

En este ejemplo de tutorial, la actividad tiene 2 entradas (archivo y datos JSON) y 1 salida (archivo).
