El proyecto de .NET Core ASP.NET que creamos en Visual Studio está vacío, por lo que cuando lo iniciamos, no se ejecuta nada. Como nuestro proyecto designAutomationSample ahora contiene una página de inicio en wwwroot/index.html deberíamos ver su contenido al iniciarlo.

Visual Studio debe enumerar todos los exploradores disponibles en el sistema para que pueda elegir uno para iniciar el proyecto. Sólo tienes que seleccionar uno y hacer clic en el icono "reproducir". El navegador debería abrirse con la página de inicio.
