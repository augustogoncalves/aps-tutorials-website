# Automatización del diseño

## Introducción

Este tutorial lo guiará a través de la creación de una aplicación web que puede cargar un archivo de entrada, cambiar sus parámetros 'ancho' y 'alto' y guardar el archivo de salida. La interfaz de usuario le permitirá definir AppBundle y actividad (como configuración inicial) y ejecutar Workitems, con un resultado de salida.

> Tanto los archivos de entrada como los de salida se guardan en depósitos OSS, puede utilizar [Visor simple](../simple-viewer/) para verlos.

<ResponsiveVideo src={PreviewUrl} />
To modify your models, you need the following steps:

Create a server << First time developer? You should start here

- Interfaz de usuario básica de la aplicación
- Prepara un plugin
- Definir una actividad
- Ejecutar elemento de trabajo
