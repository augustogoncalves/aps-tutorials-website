Si la aplicación aún se está ejecutando, reinícielo (por ejemplo, usando _Debug > Restart_,
o pulsando 'Ctrl' + 'Mayús' + 'F5'), de lo contrario, inícielo de nuevo (usando _Debug > Debugging_ Inicio,
o pulsando 'F5'). Al navegar a [http://localhost:8080/api/auth/token](http://localhost:8080/api/auth/token)
en el navegador, el servidor ahora debería responder con un objeto JSON que contenga los datos del token de acceso.
