¡Y eso es todo! Su aplicación ya está lista para la acción. Inícialo desde la línea de comandos como de costumbre:

```bash
export APS_CLIENT_ID=your-own-aps-client-id
export APS_CLIENT_SECRET=your-own-aps-client-secret
export APS_BUCKET=optional-custom-bucket-name
dotnet run
```

Y luego navegue hasta [http://localhost:8080](http://localhost:8080) en su navegador. Deberías estar
presentado con una interfaz de usuario simple, con un menú desplegable en la esquina superior derecha que eventualmente se completará
con todos los modelos disponibles en su bucket configurado y con un botón para cargar nuevos modelos.
Y tan pronto como seleccione una de las opciones del menú desplegable, se cargará el modelo correspondiente
en el visor que ocupa el resto de la página web.
