Otra parte esencial del desarrollo de software moderno, especialmente para la web, es el código fuente
Herramienta de gestión [git](https://git-scm.com). Vaya a [Descargas](https://git-scm.com/downloads)
e instale la versión más reciente para su plataforma. Como se mencionó en la sección anterior,
el instalador de Windows también viene con la CLI 'bash'.

Para comprobar si 'git' está instalado y disponible, intente ejecutar el siguiente comando en 'bash':

```bash
git --version
```
