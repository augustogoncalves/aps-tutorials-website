La interfaz de línea de comandos, una _CLI_, es una herramienta esencial a la hora de desarrollar aplicaciones web modernas.
A menudo se usa para inicializar nuevos proyectos, manejar sus dependencias o para administrar sus
código fuente. Los diferentes sistemas operativos tienen sus propias opciones (por ejemplo, en Windows hay
el [Símbolo del sistema](https://en.wikipedia.org/wiki/Cmd.exe) y [PowerShell](https://en.wikipedia.org/wiki/PowerShell),
y los sistemas basados en Unix tienen todo tipo de [shells](https://en.wikipedia.org/wiki/Unix_shell))
Pero en nuestro caso vamos a usar 'bash', que está disponible _cross-platform_.

- En sistemas basados en Unix, lo más probable es que 'bash' (o uno de sus sabores como 'zsh') esté disponible
ya, y si no, puedes conseguirlo en https://www.gnu.org/software/bash.
- En Windows, puede instalar [Git para Windows](https://gitforwindows.org) que viene incluido
con un terminal 'bash'
