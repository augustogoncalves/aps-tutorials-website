Para la codificación en sí, puedes usar el editor que más te guste.
Si busca recomendaciones, le recomendamos [Visual Studio Code](https://code.visualstudio.com)
ya que es gratuito, multiplataforma, rápido y repleto de muchas funciones excelentes.
Simplemente instale la última compilación estable para su plataforma.



:::tip

Hemos creado una extensión para Visual Studio Code que proporciona información sobre
algunos de los servicios y datos de APS directamente desde el editor:
[Herramientas de Autodesk Forge](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
Esto podría ser útil al depurar sus propias aplicaciones APS.


:::
