命令行界面（_CLI_）是开发现代 Web 应用程序时必不可少的工具。
它通常用于初始化新项目、处理其依赖项或管理其
源代码。不同的操作系统有自己的选项（例如，在 Windows 中
[命令提示符](https://en.wikipedia.org/wiki/Cmd.exe） 和 [PowerShell](https://en.wikipedia.org/wiki/PowerShell），
基于 Unix 的系统有各种各样的 [shell](https://en.wikipedia.org/wiki/Unix_shell））
但在我们的例子中，我们将使用“bash”，它_cross platform_可用。

- 在基于 Unix 的系统上，“bash”（或其风格之一，如“zsh”）最有可能可用
已经，如果没有，您可以从 https://www.gnu.org/software/bash 获得它。
- 在 Windows 上，您可以安装捆绑的 [Git for Windows](https://gitforwindows.org）
带有“bash”终端
