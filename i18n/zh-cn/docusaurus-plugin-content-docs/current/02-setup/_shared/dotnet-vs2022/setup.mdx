在我们的教程中，我们将使用 Visual Studio Community 2022。您可以下载_Visual Studio Installer_
从 [here](https://visualstudio.microsoft.com/vs/community/），然后安装所需的版本和版本
安装程序应用程序中的 Visual Studio：

![Visual Studio Installer](vs-installer.webp)

请确保包含 _ASP.NET 和 Web development_ 组件，并考虑包括
_.NET 桌面development_组件，如果您计划为桌面应用程序开发插件
例如 Revit 或 Inventor。
