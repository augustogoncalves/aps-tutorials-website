对于编码本身，欢迎您使用您最喜欢的任何编辑器。
如果要查找建议，建议使用 [Visual Studio Code](https://code.visualstudio.com）
因为它是免费的、跨平台的、快速的，并且包含许多很棒的功能。
只需为您的平台安装最新的稳定版本即可。



:::tip

我们为 Visual Studio Code 构建了一个扩展，提供对
一些 APS 服务和数据直接来自编辑器：
[Autodesk Forge 工具](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools）。
这在调试您自己的 APS 应用程序时可能很有用。


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

此扩展用于在 Visual Studio Code 编辑器中以 C# 开发 Web 应用程序：
[C# 扩展](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp）。
这在设置和调试您自己的 APS 应用程序时可能很有用。


:::
