# 定义活动

活动是可以使用指定引擎执行的操作的规范。它指定输入和输出文件的数量，以及要使用的 AppBundle 和入口点。

在本教程示例中，活动有 2 个输入（文件和 JSON 数据）和 1 个输出（文件）。

<EnvTabs
  NodeJsVsCode={ActivityNodeJs}
  DotNetVsCode={ActivityNet6}
  DotNetVs2022={ActivityNet6}
/>

现在，您可以单击“配置”（右上角），选择“AppBundle”，选择“引擎”
，然后单击 Define Activity，它应该定义并上传 appbundle 并定义
活动。结果面板（左侧）显示相应的 ID。所有其他按钮
还不工作...让我们继续前进。
