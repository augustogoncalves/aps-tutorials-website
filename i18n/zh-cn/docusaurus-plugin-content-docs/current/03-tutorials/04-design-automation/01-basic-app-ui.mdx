# 基本用户界面

该界面基于原版HTML5和“JavaScript”。对于任何服务器端，它基本上都是一样的，但有一些区别：Websocket 实现使用 socket.io （'Node.js'） 或 SignalR （'.NET 6'）。
让我们从 UI（“HTML”和“JavaScript”）文件开始。在“wwwroot”文件夹下，创建一个包含以下内容的“index.html”文件：

<EnvTabs
  NodeJsVsCode={IndexHtmlForNode}
  DotNetVsCode={IndexHtmlForNET}
  DotNetVs2022={IndexHtmlForNET}
/>

## ApsDesignAutomation.js

在“wwwroot”文件夹下创建一个“js”文件夹。然后，在“wwwroot/js”文件夹中，创建一个包含以下内容的“ApsDesignAutomation.js”文件：

<EnvTabs
  NodeJsVsCode={DAJSForNode}
  DotNetVsCode={DAJSForNET}
  DotNetVs2022={DAJSForNET}
/>
