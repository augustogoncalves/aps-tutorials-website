我们在 Visual Studio 上创建的 .NET Core ASP.NET 项目为空，因此当我们启动它时，不会运行任何内容。由于我们的 designAutomationSample 项目现在在 wwwroot/index.html 处包含一个起始页，因此我们应该在启动时看到它的内容。

Visual Studio 应列出系统中所有可用的浏览器，以便你选择一个浏览器来启动项目。只需选择一个，然后单击“播放”图标。浏览器应打开您的起始页。
