在解决方案资源管理器中，选择“designAutomationSample”项目，在“属性”下，
选择并打开“launchSettings.json”

- 将“applicationUrl”更改为“http://localhost:8080”
- 在“launchSettings.json”中，转到“profiles\designAutomationSample\environmentVariables”，添加以下环境。

```bash
ASPNETCORE_URLS:"http://localhost:8080"
APS_CLIENT_ID:"Your Id Here"
APS_CLIENT_SECRET:"Your Secret Here"
```

- 选择“designAutomationSample”项目，右键单击“添加”->“新建项”，选择“JSON”。
- 将其命名为“appsettings.user.json”。
- 添加以下配置设置，这是 [DesignAutomation SDK](https://www.nuget.org/packages/Autodesk.Forge.DesignAutomation） 所必需的
  创建 oAuth 并运行 API 请求。

```json title=appsettings.user.json
{
  "Forge": {
    "ClientId": "your client id",
    "ClientSecret": "your secret"
  }
}
```

由于这是在本地运行的，因此“sslPort”设置为 0。它应该如下所示。

```json title="launchSettings.json"
{
  "iisSettings": {
    "windowsAuthentication": false,
    "anonymousAuthentication": true,
    "iisExpress": {
      "applicationUrl": "http://localhost:8080",
      "sslPort": 0
    }
  },
  "profiles": {
    "designAutomationSample": {
      "commandName": "Project",
      "dotnetRunMessages": true,
      "launchBrowser": true,
      "applicationUrl": "http://localhost:8080",
      "environmentVariables": {
        "ASPNETCORE_ENVIRONMENT": "Development",
        "APS_CLIENT_ID": "your client id here",
        "APS_CLIENT_SECRET": "your secret here"
      }
    },
    "IIS Express": {
      "commandName": "IISExpress",
      "launchBrowser": true,
      "environmentVariables": {
        "ASPNETCORE_ENVIRONMENT": "Development"
      }
    }
  }
}
```
