# 身份验证

在此步骤中，我们将扩展服务器实现，以便它可以对自己进行身份验证
到 APS 平台，并为不同的用例生成访问令牌。



:::tip

最好生成具有更多功能的“内部”令牌（例如，
允许您在数据管理服务中创建或删除文件），这些文件将仅被使用
由服务器提供，以及具有较少功能的“公共”令牌，可以安全地与之共享
客户端逻辑。


:::
