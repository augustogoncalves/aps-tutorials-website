# 简单查看器

## 简介

在本教程中，我们将构建一个简单的 APS 应用程序，让我们
上传、转换和预览 3D 设计和 2D 图纸。

<Image img={PreviewUrl} />

我们将使用平台的以下组件：

- [认证](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview）
- [数据管理](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview）
- [模型导数](https://aps.autodesk.com/en/docs/model-derivative/v2/developers_guide/overview）
- [查看者](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview）



:::tip

如果您对从头开始构建应用程序不感兴趣，而只想尝试
并在本地运行，完整的实现可在 GitHub 上找到：
- 对于Node.js：https://github.com/autodesk-platform-services/aps-simple-viewer-nodejs
- 对于 .NET：https://github.com/autodesk-platform-services/aps-simple-viewer-dotnet


:::
