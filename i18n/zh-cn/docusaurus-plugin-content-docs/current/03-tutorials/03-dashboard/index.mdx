# 仪表板

## 简介

在本教程中，我们将增强 [Simple Viewer](../simple-viewer） 应用程序
具有类似仪表板的功能，例如，检索和聚合信息
从我们的设计元数据中，在图表和数据网格中显示结果，并控制
基于与这些新视觉对象的交互的查看器状态。

All the functionality in this tutorial is implemented as viewer extensions, and will only use
the viewer UI, so you're free to use any APS application you want as your starting point.
Note however that the names of folders, and the way you import JavaScript files into your
web application, might be a little bit different.



:::caution

本教程代码的某些部分假定设计元素包含特定属性
例如 _Volume_、_Level_ 或 _Price_。如果您有想要使用的特定设计，则可以
当然修改代码以使用其他属性，否则我们建议使用官方
[Revit 2022 示例项目文件](https://knowledge.autodesk.com/support/revit/getting-started/caas/CloudHelp/cloudhelp/2022/ENU/Revit-GetStarted/files/GUID-61EF2F22-3A1F-4317-B925-1E85F138BE88-htm.html）
这些属性始终存在。


:::
