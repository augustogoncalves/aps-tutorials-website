# Hubs 浏览器

## 简介

在本教程中，我们将构建一个 APS 应用程序，让我们浏览内容
其他基于 APS 的应用程序（如 [BIM 360 Docs](https://www.autodesk.com/bim-360/construction-document-management-software/） 等经过身份验证的用户，
[Autodesk 文档](https://www.autodesk.com/products/autodesk-docs/overview） 或 [Fusion Team](https://www.autodesk.com/products/fusion-360/overview）。



:::note

确保 [预配访问权限](../../01-getting-started/index.mdx#provision-access-in-other-products） 添加到您的 ACC 或 BIM360 帐户之一
以便应用程序可以看到您的项目、文件夹和文档。


:::



<Image img={PreviewUrl} />

我们将使用平台的以下组件：

- [认证](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview）
- [数据管理](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview）
- [查看者](https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/overview）



:::tip

如果您对从头开始构建应用程序不感兴趣，而只想尝试
并在本地运行，完整的实现可在 GitHub 上找到：
- https://github.com/autodesk-platform-services/aps-hubs-browser-nodejs
- https://github.com/autodesk-platform-services/aps-hubs-browser-dotnet


:::
