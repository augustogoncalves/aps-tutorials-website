像往常一样从 Visual Studio Code 启动（或重启）应用，然后导航到
[http://localhost:8080/api/hubs](http://localhost:8080/api/hubs） 在浏览器中。
服务器应使用您有权访问的所有中心的 JSON 列表进行响应。
尝试复制其中一个集线器的 ID，并将其用于另一个地址：http://localhost:8080/api/hubs/your-hub-id/projects。
在这种情况下，服务器应用程序应使用所有项目的 JSON 列表进行响应
在指定的中心下可用。
