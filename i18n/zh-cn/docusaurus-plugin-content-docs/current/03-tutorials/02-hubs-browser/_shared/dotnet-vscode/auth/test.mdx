如果应用程序仍在运行，请重新启动它（例如，使用 _Run > 重新启动 Debugging_，
或单击绿色的重新启动图标），否则再次启动它（使用_Run >开始Debugging_，
或按“F5”）。

导航到 [http://localhost:8080/api/auth/login](http://localhost:8080/api/auth/login） 时
在浏览器中，您应该被重定向到Autodesk登录页面，登录后，您应该
重定向回您的应用程序，现在只需返回 404。这是意料之中的，因为我们没有
实现了“GET /”端点。但是，如果您使用浏览器开发工具并浏览 cookie
由浏览器存储为“localhost”源，您会注意到该应用程序已经
将身份验证数据存储在那里。
