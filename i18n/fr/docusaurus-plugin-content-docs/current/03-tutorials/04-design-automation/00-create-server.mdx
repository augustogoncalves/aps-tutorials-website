# Créer un serveur

Votre 'ID client' et votre 'Secret' doivent être protégés et gardés confidentiels car tous vos fichiers seront liés à votre compte. Pour une application web, conservez-la sur votre serveur. Cette section montre comment préparer la création d’un serveur de développement local.

Veuillez consulter [Configuration de l’environnement](.. /.. /setup/) pour les logiciels requis.
