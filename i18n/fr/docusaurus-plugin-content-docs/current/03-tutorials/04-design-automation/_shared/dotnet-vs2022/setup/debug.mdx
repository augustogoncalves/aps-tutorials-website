Le projet .NET Core ASP.NET que nous avons créé sur Visual Studio est vide, donc lorsque nous le démarrons, rien ne s’exécute. Comme notre projet designAutomationSample contient maintenant une page de démarrage à wwwroot/index.html nous devrions voir son contenu au lancement.

Visual Studio doit répertorier tous les navigateurs disponibles dans le système pour que vous en choisissiez un pour démarrer votre projet. Il suffit d’en sélectionner un et de cliquer sur l’icône « jouer ». Le navigateur doit s’ouvrir avec votre page d’accueil.
