# Automatisation de la conception

## Présentation

Ce tutoriel vous guidera dans la création d’une application web capable de télécharger un fichier d’entrée, de modifier ses paramètres 'width' et 'height' et d’enregistrer le fichier de sortie. L’interface utilisateur vous permet de définir AppBundle et Activity (en tant que configuration initiale) et d’exécuter des éléments de travail, avec un résultat de sortie.

> Les fichiers d’entrée et de sortie sont enregistrés dans les compartiments OSS, vous pouvez utiliser [Simple Viewer](.. /simple-viewer/) pour les visualiser.

- Définir une activité
