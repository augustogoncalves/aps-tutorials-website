Si l’application est toujours en cours d’exécution, redémarrez-la (par exemple, à l’aide de _Run > Redémarrer Debugging_,
soit en cliquant sur l’icône verte de redémarrage), sinon redémarrez-le (en utilisant _Run > Démarrer Debugging_,
ou en appuyant sur 'F5'). Lorsque vous accédez à [http://localhost:8080/api/auth/token](http://localhost:8080/api/auth/token)
dans le navigateur, le serveur doit maintenant répondre avec un objet JSON contenant les données du jeton d’accès.
