Et c’est tout ! Votre application est maintenant prête à être mise en œuvre. Démarrez-le à partir de Visual Studio, puis naviguez
à [http://localhost:8080](http://localhost:8080) dans votre navigateur. On devrait vous présenter une interface utilisateur simple,
avec une liste déroulante dans le coin supérieur droit qui finira par être remplie de tous les modèles disponibles dans votre
compartimenté configuré, et avec un bouton pour télécharger de nouveaux modèles. Et dès que vous sélectionnez l’une des options
Dans la liste déroulante, le modèle correspondant sera chargé dans la visionneuse occupant le reste de la page Web.
