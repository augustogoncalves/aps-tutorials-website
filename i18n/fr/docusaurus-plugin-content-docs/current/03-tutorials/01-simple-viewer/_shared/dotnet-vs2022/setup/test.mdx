Démarrez l’application à partir de Visual Studio (par exemple, via le _Debug > Démarrer Debugging_
ou en appuyant sur 'F5'), et il ouvrira automatiquement [http://localhost:8080](http://localhost:8080)
dans votre navigateur. Vous devriez obtenir une réponse 404 car nous n’avons pas encore implémenté de logique de serveur.
Ce sera l’objectif de la prochaine étape.
