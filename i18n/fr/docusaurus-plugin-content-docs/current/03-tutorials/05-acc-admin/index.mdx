# Administrateur ACC

## Présentation



:::note

Seul **Node.js** est disponible pour le tutoriel.


:::




In this tutorial we're going to build an APS application that will let us browse Hubs and Projects
of our authenticated user in [Autodesk Construction Cloud](https://www.autodesk.com/products/autodesk-docs/overview),
list all the information of projects and project users, you can also export these information to a CSV
file easily, but most important, the tutorial will show case how to import multiple projects into a selected account
based on a template project list csv file, or import multiple users into a selected project from user list template 
csv file.



:::note

Assurez-vous de [provisionner l’accès](.. /.. /01-getting-started/index.mdx#provision-access-in-other-products) à vos comptes ACC
afin que l’application puisse voir vos projets.


:::



<Image img={PreviewUrl} />

Nous utiliserons les composants suivants de la plateforme :

- [Authentification](https://aps.autodesk.com/en/docs/oauth/v2/developers_guide/overview)
- [Gestion des données](https://aps.autodesk.com/en/docs/data/v2/developers_guide/overview)
- [Administrateur Autodesk Construction Cloud](https://aps.autodesk.com/en/docs/acc/v1/overview/field-guide/admin/)



:::tip

Si vous n’êtes pas intéressé par la création de l’application à partir de zéro et que vous voulez juste essayer
et exécutez-le localement, l’implémentation complète est disponible sur GitHub :
- https://github.com/autodesk-platform-services/aps-acc-admin-tutorial-nodejs


:::
