## Configuration de l’application

Notre application aura besoin de quelques paramètres de configuration pour fonctionner correctement, par exemple,
les informations d’identification de notre application APS pour communiquer avec les services Autodesk Platform ou l’URL de rappel
où nos utilisateurs seront redirigés pendant le [workflow d’authentification à 3 pattes](https://aps.autodesk.com/en/docs/oauth/v2/tutorials/get-3-legged-token/).
Nous allons transmettre ces paramètres à l’application serveur à l’aide de variables d’environnement.
