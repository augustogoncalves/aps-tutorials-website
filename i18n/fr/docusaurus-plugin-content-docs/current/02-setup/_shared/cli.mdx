L’interface en ligne de commande, une _CLI_, est un outil essentiel lors du développement d’applications Web modernes.
Il est souvent utilisé pour initialiser de nouveaux projets, gérer leurs dépendances ou gérer leur
code source. Différents systèmes d’exploitation ont leurs propres options (par exemple, dans Windows, il y a
l'[invite de commande](https://en.wikipedia.org/wiki/Cmd.exe) et [PowerShell](https://en.wikipedia.org/wiki/PowerShell),
et les systèmes basés sur Unix ont toutes sortes de [shells](https://en.wikipedia.org/wiki/Unix_shell))
Mais dans notre cas, nous allons utiliser 'bash' qui est disponible _cross-platform_.

- Sur les systèmes basés sur Unix, 'bash' (ou l’une de ses saveurs comme 'zsh') est très probablement disponible
Déjà, et sinon, vous pouvez l’obtenir auprès de https://www.gnu.org/software/bash.
- Sous Windows, vous pouvez installer [Git pour Windows](https://gitforwindows.org) qui est fourni
avec un terminal 'bash'
