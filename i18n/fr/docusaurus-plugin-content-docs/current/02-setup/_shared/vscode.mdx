Pour le codage lui-même, vous pouvez utiliser l’éditeur que vous préférez.
Si vous recherchez des recommandations, nous vous recommandons [Visual Studio Code](https://code.visualstudio.com)
car il est gratuit, multiplateforme, rapide et doté de nombreuses fonctionnalités intéressantes.
Il vous suffit d’installer la dernière version stable pour votre plateforme.



:::tip

Nous avons créé une extension pour Visual Studio Code qui fournit des informations sur
certains des services et données APS directement de l’éditeur :
[Outils Autodesk Forge](https://marketplace.visualstudio.com/items?itemName=petrbroz.vscode-forge-tools).
Cela peut être utile lors du débogage de vos propres applications APS.


:::




Then you need to install [C# Extension](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) (if don't have it yet) for Visual Studio Code. To install, click on extensions tab on the left sidebar of your Visual Studio Code, then search for ```C#``` and click install as shown in the image below.



:::info

Cette extension est utilisée pour développer des applications web en C# dans l’éditeur Visual Studio Code :
[Extension C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp).
Cela peut être utile lors de la configuration et du débogage de vos propres applications APS.


:::
