Dans nos tutoriels, nous utiliserons Visual Studio Community 2022. Vous pouvez télécharger _Visual Studio Installer_
à partir de [ici](https://visualstudio.microsoft.com/vs/community/), puis installez la version et l’édition souhaitées
de Visual Studio à partir de l’application d’installation :

![Visual Studio Installer](vs-installer.webp)

Assurez-vous d’inclure le composant _ASP.NET et Web development_, et envisagez d’inclure
le composant _.NET desktop development_ également si vous prévoyez de développer des plug-ins pour les applications de bureau
tels que Revit ou Inventor.

![Visual Studio Components](vs-components.webp)
